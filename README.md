# Windows11X

A debloated and custom iso built off of Windows 11
Currently in beta.

# Preinstalled apps
Windows 11X comes with the following:
-Win10 Widgets (Rainmeter)
-Windhawk
-ExplorerPatcher
-Chrome (replaces Edge)
-Dotnet SDK
-Winaero Tweaker
And many more to come!
